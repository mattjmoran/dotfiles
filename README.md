# dotfiles

Cross-platform, cross-shell dotfiles and system configuration.

## Table of Contents

- [Install](#install)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Install

```sh
mise install
```

Requires [mise](https://mise.jdx.dev/getting-started.html).

## Usage

```sh
mise run lint       # run all linters
mise run fmt        # auto-format files
mise run update     # update all tools and plugins
```
